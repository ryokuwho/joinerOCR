# Joiner OCR

複数のスクリーンショットをスマートに繋ぎ合わせ、OCR（文字認識）によって精密な位置合わせを行う単一HTMLツールです。

## 🚀 主な機能

- **Smart Align (自動結合)**: Tesseract.js（OCR）を使用して、画像内のテキストや日付を解析。重複箇所を自動で検出し、ピクセル単位で正確に結合します。
- **完全ローカル処理**: すべての画像処理と文字認識はブラウザ上で行われます。サーバーにデータが送信されることはなく、プライバシーが守られます。
- **ピクセル・パーフェクト**: スクリーンショットの鮮明さを損なわないよう、サブピクセル描画（ボケ）を排除した高品質な描画・保存が可能です。
- **直感的なUI**: 
    - ドラッグ＆ドロップによる結合位置の微調整。
    - 画像の順序変更や個別の削除。
    - 自由なズーム操作による詳細確認。
- **ポータブル**: インストール不要。`joinerOCR.html` ひとつで動作します。

## 📖 使い方

1. **プロジェクトの起動**: `joinerOCR.html` をブラウザ（Chrome, Edge, Safari等）で開きます。
2. **画像の読み込み**: 「追加」ボタンから、結合したいスクリーンショットを複数選択します。
3. **自動位置合わせ**: 「Smart Align」をクリックします。解析が完了すると、画像が自動的に繋がります。
4. **微調整**: プレビュー上の画像をドラッグすることで、垂直方向のズームや重なり具合を細かく調整できます。
5. **保存**: 「保存」ボタンをクリックすると、結合結果のプレビューを含む保存ダイアログ（ボトムシート）が表示されます。内容を確認し、「保存 (ダウンロード)」をクリックして画像を保存してください。

## 🛠 技術構成

- **Core**: HTML5, JavaScript (ES6+)
- **CSS**: Tailwind CSS (CDN版)
- **OCR**: [Tesseract.js](https://tesseract.projectnaptha.com/)
- **Design Area**: モバイル・デスクトップ両対応のレスポンシブデザイン

## 🔒 プライバシーについて

本ツールは「完全オフライン対応可能」です。OCRエンジン（Tesseract.js）の初回読み込み時以外、ネットワーク通信は発生せず、ユーザーの画像データが外部に流出することはありません。
